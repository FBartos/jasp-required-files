makeGridLines {JASPgraphs}R Documentation

custom Gridlines for ggplot objects

Description

custom Gridlines for ggplot objects

Usage

makeGridLines(x, xend, y, size = 0.85, ...)

Arguments

x

Left bound of gridline.

xend

Right bound of gridline.

y

height of gridline.

size

size of the line.

...

Further arguments to geom_segment, e.g., colour.

linetypes

solid, dashed, dotted, etc.

Details

This function exists only when gridlines need to exist at specific locations, for example from x1 to x2 but don't extend further than x2. Otherwise, use the build in functionality inside theme. This function is a wrapper around geom_segment.

Value

a ggproto object.
