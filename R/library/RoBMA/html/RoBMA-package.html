RoBMA: Robust Bayesian Meta-Analyses

Description

The package estimates an ensemble of Bayesian model-averaged meta-analytical models (assuming either presence or absence of the effect, heterogeneity, and publication bias). The ensemble uses Bayes Factors to test for presence of absence of the individual components and model-averages parameter estimates based on posterior model probabilities. The user can define a wide range of non-informative or informative priors for the effect size, heterogeneity, and weight functions. The package provides convenient functions for summary, visualizations, and fit diagnostics.
